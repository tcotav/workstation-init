
## New Workstations

Assuming you have a directory `~/github` set up where you'll dump this stuff

    cd ~/github
    git clone git@github.com:tcotav/workstation-init.git
    cd workstation-init
    ln -s vim ~/.vim
    ln -s screenrc ~/.screenrc
    ln -s bashrc ~/.bashrc

## Short term VMs
### simple install

    git clone git@github.com:tcotav/workstation-init.git
    cd workstation-init
    ./install.sh
